Support

Need help with an XZDDev open source app or project? You can contact support for bug reports, feature requests, documentation issues, or general project questions.

Contact Support

For support requests, email us and include only the information you are comfortable sharing. XZDDev projects collect zero user data, so we may need you to voluntarily describe the issue if you want help troubleshooting it.

acdelectrical@icloud.com

Helpful Information

If relevant, the following details can help with troubleshooting. Please do not include passwords, private keys, access tokens, sensitive personal information, or anything you do not want to share.

  • The project or app name
  • Your device model or operating system
  • The app, browser, or project version you are using
  • A short description of the issue
  • Steps to reproduce the problem
  • Screenshots if relevant

Open Source Issues

For open source projects, you may also report bugs, request features, or contribute fixes through the relevant GitHub or GitLab repository. Any information you post on those platforms is handled by that platform and may be publicly visible.

Privacy

XZDDev projects are designed to collect zero user data. Support emails are used only to read, respond to, investigate, and resolve the request. They are not used for analytics, advertising, profiling, or resale.

Olilo Status is operated separately. For Olilo Status support, visit: Olilo Status support repository
